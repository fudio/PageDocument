5. Flex
	*flex:1 // có thể 1/2
	Để sử dụng phải gắng cho các "View Con"
	Dãn toàn màng hình.(Auto Layout)
	// Thuộc tính.
	1.
	Thuộc tính. 
	- flexDirection:'row'|'Column';
	Tác dụng.
	- Xác định sự "sắp xếp" theo hướng.
	Mặc định
	- Column
	2. 
	Thuộc tính 
 	- justifyContent:'flex-start'|'center'|'flex-end'|'space-between'|'space-around'; 
	Tác dụng: 
	- Canh theo trục. cho "từng" Item
	Default.
	- flex-start
	3.
	Thuộc tính 
 	- alignItems:'flex-start'|'center'|'flex-end'|'stretch'; 
	Tác dụng: 
	- Canh theo trục. cho "Toàn bộ" Item
	Default.
	- stretch
	4.
	Thuộc tính 
 	- alignSelf:'flex-start'|'center'|'flex-end'|'stretch'; 
	Tác dụng: 
	- Canh theo trục. cho "1" Item. Nhớ để vào Item cần canh
	Default.
	- stretch

	npm install react-navigation-stack
	2. Thêm 2 thư viện
		2.1. createStackNavigator from 'react-navigation-stack';
		2.2. createAppContainer from 'react-navigation';
	3. Cấu hình
	- Thêm 1 thư mục routes và tạo ra 1 file

	- Tạo 1 biến const screens ={ Home:screen:Home}

	- Tạo 1 biến const HomeStack = createStackNavigator(screens);
	- Xuất nó ra export default createAppContainer(HomeStack);
